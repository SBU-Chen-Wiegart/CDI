# CDI
code for CDI phase retrieval
* Align_marker.ipynb: CDI resutls for align marker
* Cu_ptycho_linearshift.ipynb: Linear shift of the final ptycho to serve as different initial guess for CDI
* Cu_ptycho_linearcombination.ipynb: Linear combination of the ptycho before and after as the initial guess for CDI
## Steps
* Crop the diffraction pattern to the same size as the probe
* Flat field correction to confirm the sample position
* Crop the inital guess from ptychogrpahy
* ER, HIO and ML for phase retrieval
